# IWXXM

__ICAO Meteorological Information Exchange Model__

This repository contains an [OpenAPI]([OpenApi](https://spec.openapis.org/oas/v3.1.0.html)) model of [IWXXM, version 2025-2](https://schemas.wmo.int/iwxxm/2025-2/). It is based on the [IWXXM Logical (UML) Model](https://schemas.wmo.int/iwxxm/2025-2/html/), and as such ignores much of the detail contained in the IWXXM XML schemas.

## Acknowledgement

The material presented in this repository is based on, and borrows from, the IWXXM UML model.
